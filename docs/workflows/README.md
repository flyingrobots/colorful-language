# Contributor workflows

Contributor workflows describe repository operations rather than Colorful's
product behavior. They are current operational contracts for maintainers and
agents.

Use this area for procedures such as release preparation, publishing,
repository maintenance, and other recurring work that changes how the project is
operated.

Product and user-facing concepts still belong in [`../topics/`](../topics/).

## Workflows
